# 脚本目录

本目录包含项目相关的工具脚本和辅助文件。

## 脚本说明

### 构建和打包脚本
- `build_static.py` - 构建静态文件
- `cloud_hosting_package.py` - 云托管打包脚本
- `create_lightweight_package.py` - 创建轻量级打包
- `create_upload_package.py` - 创建上传打包

### 修复和工具脚本
- `fix_stage_names.js` - 修复阶段名称
- `fix_streamlit_cloudbase.py` - 修复 Streamlit CloudBase 配置
- `permission_system_guide.py` - 权限系统指南脚本

## 使用说明

运行这些脚本前，请确保：
1. 已安装必要的依赖
2. 已配置正确的环境变量
3. 已阅读相关文档
